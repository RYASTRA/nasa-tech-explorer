Software for an Engineering Exposure Time Calculator for Coronagraphs

Case number
NPO-53377-1
NASA center
JPL
Category
data and image processing
First seen
2026-07-20

This software determines the required detector settings for engineering images with the Nancy Grace Roman Space Telescope Coronagraph Instrument's Exoplanetary Systems Camera (EXCAM), an EMCCD, for a given astronomical target. The example configuration files for Roman CGI do not contain any export controlled information.
