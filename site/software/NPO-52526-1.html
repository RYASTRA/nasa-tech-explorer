Machine Learning Key Utilities (MLky)

NASA center
JPL

MLky is a Python configuration software package for data scientists. It is designed to be highly configurable and flexible to be able to adapt to the evolving needs of data scientists. This facilitates research and pipeline development as well as provides a framework for software packages to step up its configuration from a flexible prototype to a rigid release candidate
