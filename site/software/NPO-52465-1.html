Trace gas profile generation software (GINPUT)

Case number
NPO-52465-1
NASA center
JPL
Category
data and image processing
First seen
2026-07-20

Generates vertical profiles of greenhouse gases and other trace gases. It primarily generates these profiles using only long term trends from NOAA observatories and correlations with meteorological variables. This allows the profiles to be independent of models, which in turn means satellite retrievals that use this software's profiles as prior information are a better independent comparison for said models.
