The Health and Safety application (HS) is a core Flight System (cFS) application that is a plug in to the Core Flight Executive (cFE) component of the cFS. The HS application provides functionality for Application Monitoring, Event Monitoring, Hardware Watchdog Servicing, Execution Counter Reporting (optional), and CPU Aliveness Indication (via UART).
