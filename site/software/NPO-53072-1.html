Probabilistic emulation for land surface model ensembles

Case number
NPO-53072-1
NASA center
JPL
Category
data and image processing
First seen
2026-07-20

This software provides tools for statistical emulation of spatial fields from land surface models (LSMs). The tools include pre-processing datasets output from LSM ensemble simulations to facilitate integration with a package for fitting the emulator and simulating new realizations of spatial fields using the results.
