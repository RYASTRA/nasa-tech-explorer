LAURA simulates hypersonic chemically reacting fluid dynamics. In this version, a new grid adaptation allows for multiple grid blocks to be stacked perpendicular to a vehicle. Previously with the LAURA software, only a single grid block could be adapted in the direction perpendicular to a simulated vehicle.
