Surface Robotics Mockups Description

JSC

This NTR is a software package containing the descriptions and simulation files for mock-ups for testing robotic capabilities for future lunar missions. The mock-ups include a 'task trainer' (the interior), a 40in-by-40in hatch, and a 40in-by-60in hatch for a pressurized rover.
