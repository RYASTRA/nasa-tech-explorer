The Parallel Observing System Simulation Experiment (ParOSSE) code serves as a flexible and extensible framework for the evaluation of science benefit for current and new observing systems. It is designed to rapidly explore the information content in diverse measurement concepts using simulated reference data (a nature run), an instrument simulator, and a set of analysis and plotting functions. This version is tailored specifically for the needs of the National Oceanic and Atmospheric Administration (NOAA).
