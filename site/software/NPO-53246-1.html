This python notebook calculates instrument radiometric performance for the Carbon-I mission. It is adapted from code developed for similar calculations for the EMIT and SBG-VSWIR instruments. The Carbon-I instrument is a Dyson-layout imaging spectrometer that measures spectra in the 2-2.4 micron range. We first create top-of-atmosphere radiances for a predefined "reference scene" and then assess instrument performance in terms of noise sources, signal levels, and total SNR. The code below uses the functions in instrument.py for radiometric calculations.
