The Ingenium electronic procedure system is a platform for developing and executing test procedures on complex systems. It provides numerous advantages over existing test procedure methods including a standard interface (visually and electronically), directly executable steps, implementation of redline policies, procedure authoring aids, and the ability to rapidly switch from manual to automatic test execution. It currently supports the Europa Clipper and Psyche projects for system testbeds, simulators, and in Assembly, Test, and Launch Operations.
