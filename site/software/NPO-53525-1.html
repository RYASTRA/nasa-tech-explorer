Testbed Asset Configuration for Operations (TACO)

Case number
NPO-53525-1
NASA center
JPL
Category
aeronautics
First seen
2026-07-20

TACO is a full stack web application that provides an interface and life-of-mission database to specify and track the configuration of hardware and software spacecraft components in validation and verification (V&V) and operational venues. It provides current status, snapshots, histories, and administrative interfaces for end user operators and administrators.
