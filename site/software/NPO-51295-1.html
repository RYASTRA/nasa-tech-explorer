VegMapper is an open-source package build in the R language, that employ Bayesian algorithms to generate the presence probability for a user-defined class. It automates and streamlines all GIS operations to integrate remote sensing imagery with in-situ data. The package is flexible and can accept a variable number of predictor variables and in-situ points.
