Python suite for International Radio Regulations allocation tables parsing and diagnosis.

The software ingests radio frequency allocation tables from the ITU-R Radio Regulations and (previous versions of) the FCC frequency allocations tables. This information is stored in an internal database (using the python intervaltree pacakge), and can be queried (e.g., find all the allocations to the Radio Astronomy Service that are subject to footnote RR 5.149 protection.). The software also supports generation of plots showing allocations in a given spectral band, with annotations customizable by the user.
