mux is a collection of container-based services for interacting with hardware commonly used on adaptive optics instruments for ground-based telescopes. Containerized architecture allowing only the exact features a system needs to be deployed. Similar software exists at the python module level. We improve on this by providing containerized REST API endpoints for interacting with back end hardware.
