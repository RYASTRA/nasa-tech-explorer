OptoMMP Driver for LabVIEW

The OptoMMP Driver for LabVIEW enables LabVIEW programs to communicate with Opto22 devices, such as SNAP-PAC industrial monitoring and control systems, using the Opto22 Memory Mapped Protocol (OptoMMP). The software is a native LabVIEW driver, with no external dependencies other than LabVIEW itself, and is capable of communicating with many Opto22 devices in parallel at polling rates up to 80 Hz. The driver is intended to be used in test facility data acquisition systems to collect sensor measurements, monitor facility status, and/or control test hardware. The OptoMMP Driver for LabVIEW is a LabVIEW library modeled after typical LabVIEW instrument driver libraries, and can be used by LabVIEW developers to communicate with Opto22 devices. It has standard error in and error out behavior, except where noted in the documentation. It requires no dependencies outside of LabVIEW itself.
