The autoNGC/cFS Interface for GEONS (ngc_geons) allows the Goddard Enhanced Onboard Navigation System (GEONS) software library to be used with the autonomous Navigation, Guidance and Control (autoNGC) flight software system through an app for the core Flight System (cFS).
