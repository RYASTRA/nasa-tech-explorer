The software has the following functions: 1) Create training data from JunoCam and Hubble Space Telescope in the form of segments from flattened maps2) Train a deep learning model for photometric calibration of JunoCam3) Provide a set of metrics to evaluate the calibration4) Post process the data by creating global mosaics of the calibrated segments 5) Provide tools to convert the data to PDS compliant formats
