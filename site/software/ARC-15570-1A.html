Model-Based Diagnosis Engine For Stochastic Hybrid Systems (HyDE)

Case number
ARC-15570-1A
NASA center
ARC
Category
autonomous systems
First seen

HyDE is a model-based diagnostic engine capable of detecting and isolating discrete (possibly multiple) faults in physical systems. The current version of HyDE has been implemented in C++. Please visit the following URL for more information: https://ti.arc.nasa.gov/m/project/hyde/HyDE_Summary.pdf
