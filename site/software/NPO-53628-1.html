Time series data visualization in 3D is a particularly challenging problem. Many solutions to this have been put forward however most assume a data schema that is difficult to generalize. This software looks to a take an existing 3D visualization platform, Honeycomb, and abstract the data handling from the actual visualization to allow plugging in a wide variety of data sources into the viewer. This software integrates with Grafana, an open source data observability platform that itself integrates with a wide-variety of database backends. This allows the Honeycomb viewer to focus on data visualization rather than data management/querying. It also opens avenues for easy integration with other tools/software such as Foxglove and ROS.
