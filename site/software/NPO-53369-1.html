Veloserver is a geospatial data visualization server designed to dynamically return velocity data such as winds and ocean currents from weather models and other datasets. It returns the data in formats optimized for visualizing in a web client including animated vector streamlines via gribjson. It includes built-in caching to reduce requests to external data sources and deliver quick results. It was originally developed for the NASA Earth Surface Mineral Dust Source Investigation (EMIT) mission for use by the Multi-Mission Geographic Information System (MMGIS) software.
