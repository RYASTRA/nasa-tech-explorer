Enabling Time Sensitive Networking into the Space Ecosystem

Case number
NPO-53790-1
NASA center
JPL
Category
aeronautics
First seen
2026-07-20

This software accomplishes the following:* Configures a dual-fault tolerant TSN network* Generates traffic over a TSN network in flight-like scenarios* Implements a fault-tolerant time distribution algorithm* Performs TSN to EtherCAT bridge functionality* Demonstrates a remote data concentrator over a TSN network* Demonstrates remote DMA over a fault tolerant networkConfiguration and operation of a fault tolerant TSN network for space systems has not previouslybeen done. Our approach aims to provide the above functionality using framework and OS agnosticmethods. This allows future missions and organizations inside and outside JPL to use this software to build their own space applications.
