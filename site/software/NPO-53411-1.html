This software package includes the code for the ECOSTRESS Collection 2 Level 2 Land Surface Temperature and Emissivity (LSTE) Product Generating Software (PGE). It was designed for use by the ECOSTRESS mission as a precursor for the Surface Biology and Geology (SBG) mission. This software provides estimates of LSTE and cloud mask data acquired by the ECOSTRESS radiometer. This package can be utilized for remote sensing research in Jupyter notebooks and deployed for operations in data processing pipelines. This software is being released according to the SPD-41 Open-Science requirements of NASA-funded projects.
