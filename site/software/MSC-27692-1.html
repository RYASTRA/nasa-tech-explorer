iMETRO Robot Operating System Packages

Case number
MSC-27692-1
NASA center
JSC
Category
autonomous systems
First seen
2026-07-20

This NTR is a set of software packages to integrate COTS hardware systems with Robot Operating System (ROS) software packages. These interfaces will allow the study and adaptation of terrestrial robotics technologies and the development of new technologies for non-terrestrial robotics. These systems include a horizontal rail, vertical lift, collaborative robotic arms, robotic grippers, sensors, and mounting hardware.
