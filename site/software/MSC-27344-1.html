The JSC Engineering Orbital Dynamics (JEOD) Software Package contains a set of numerical mathematical models that provide vehicle or vehicles trajectory generation by the solution of a set of numerical dynamical models. These models are comprised of an Environment model representing the gravitational and non-gravitational forces and torques acting on the vehicle or vehicles, Dynamics models for processing and numerically integrating the equations of motion, Interaction models representing interaction with the environment and a set of mathematical and orbital dynamics Utilities.
