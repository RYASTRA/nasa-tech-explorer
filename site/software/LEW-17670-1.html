Time-Accurate Quasi-One Dimensional Reactive Code For Design And Analysis Of Gasdynamic-Based Propulsion Systems

Case number
LEW-17670-1
NASA center
GRC
Category
propulsion
First seen
2026-07-20

A time-accurate, quasi-one-dimensional, reactive CFD code for design and analysis of gasdynamic-based propulsion systems has been developed. These include, but are not limited to, wave rotor based internal and external combustion systems, Pulse Detonation Engines (PDEs), and deflagrative pulse combustors. The code utilizes established numerical techniques and simplified loss-models to accurately simulate system flow fields. It contains robust boundary condition procedures which yield limit cycle behavior even in extreme off-design scenarios. The simplified quasi-one-dimensional nature of the code yields rapid solutions using modest computational resources. This allows for timely parametric analysis and preliminary design of conceptual systems.
