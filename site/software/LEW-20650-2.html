The NASA Multiscale Analysis Tool (NASMAT) serves as a state-of-the-art, plug and play, software package which utilizes multiscale recursive micromechanics as a platform for massively multiscale modeling of hierarchical materials and structures subjected to thermomechanical loads on high performance computing systems. Version 5.0 includes new features and improvements from previous versions including fatigue, interfacial debonding, non-periodic boundary conditions, coupled multi-physics, state dependent microstructure and property evolution, improvements to dry fabric module, and finite element method based micromechanics.
