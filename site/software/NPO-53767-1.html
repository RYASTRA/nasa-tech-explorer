Autonomous Underwater Vehicle Simulation

Case number
NPO-53767-1
NASA center
JPL
Category
aeronautics
First seen
2026-07-20

The software is a simulation environment for autonomous underwater vehicles that simulates vehicle motion, navigation, and sensors for the testing of autonomous planning behaviors for exploration. This software also focuses specifically on tools for testing autonomous planning algorithms by removing complex dynamic simulations that other tool provide as they are not needed for the type of testing that this software is indented for.The primary problem that is being solved is the testing and evaluation of autonomous planning algorithms in the steps before real-world deployments. Real-world testing is expensive and not many opportunities are available, so it is important to do as much simulation testing beforehand to insure success. This simulation allows for that to happen.
