The BP Core Flight System (cFS) application uses the Bundle Protocol Library (bplib) to implement Delay/Disruption Tolerant Networking Bundle Protocol (BP) version 7. The application runs on the Core Flight Executive (cFE) and provides out of the box integration with the cFS CFDP application (CF). The implementation conforms to the CCSDS BPv7 Orange Book draft and RFC 9171. The software includes test tools used to verify and validate the BPv7 cFS application.The software is divided into three repositories:https://github.com/nasa/bplibhttps://github.com/nasa/bphttps://github.com/nasa/dtn-tools/
