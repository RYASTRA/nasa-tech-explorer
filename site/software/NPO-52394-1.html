The Estimating the Circulation and Climate of the Ocean (ECCO) consortium estimates ocean circulation and its role in climate - combining state-of-the-art ocean circulation models with global ocean data sets. Data visualizations of ECCO help scientists better understand these models and provide comparisons with other data sets. The ECCO Interactive Solution Evaluator is a web application that allows users to compare data visualizations of ECCO data fields with those from other data sources side-by-side. The application includes a map interface to select a region of interest and a charting tool to plot data for comparisons.
