GDAL Plugin for NISAR HDF5 data products with Cloud-Optimized Access

Case number
NPO-53509-1
NASA center
JPL
Category
data and image processing
First seen
2026-07-20

The NISAR GDAL driver solves the critical problem of making complex, multi-layered NISAR HDF5 product data easily accessible and usable within the broad ecosystem of GDAL-supported geospatial software. It bridges the gap between the specialized HDF5 file structure and the standardized raster data model expected by GIS applications by intelligently discovering and presenting relevant raster layers as distinct subdatasets, correctly interpreting and applying NISAR-specific georeferencing (including projection and geotransform from embedded coordinate/spacing datasets), and extracting file-level and dataset-specific metadata into standard GDAL domains.
