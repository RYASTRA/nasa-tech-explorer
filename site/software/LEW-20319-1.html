The Hypersonic Aerothermodynamics Particle (HAP) code is a software tool for numerical simulation of high-speed gas flows, using shared memory parallelization and dynamically adaptive Cartesian grids. HAP is a framework built around the direct simulation Monte Carlo (DSMC) method, which is a stochastic technique for simulation of gas flows involving strong nonequilibrium among translational and/or internal (rotational, vibrational and electronic) energy modes. Unique features of HAP include capabilities for hybrid or multiscale simulation of mixed rarefied/continuum flows, as well as integrated capabilities for design optimization, uncertainty quantification and global sensitivity analysis.
