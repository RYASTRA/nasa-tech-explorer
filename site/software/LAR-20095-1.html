OVERFLOW 2.4D Overset Grid Computational Fluid Dynamics Flow Solver with Moving Body Capability (OVERFLOW)

OVERFLOW 2.4D is a computer code for simulating viscous, compressible fluid flow about complex aerodynamic configurations. Thetechnology solves the Reynolds-averaged Navier-Stokes equations using structured, overset computational grids. It includes the capability for simulating multiple moving bodiesacting under prescribed or aerodynamically forced motion. OVERFLOW 2 is a merge of the previously developed OVERFLOW 1.8 and OVERFLOW-D codes.
