Hazards Analysis Management Tool (HAMT)

Case number
GSC-16846-1
NASA center
GSFC
Category
autonomous systems
First seen
2026-07-20

HAMT uses a single software tool to increase the efficiency and effectiveness of hazard analyses. Composed of a Microsoft Access front-end (that contains the user interface) paired with a Microsoft Access back-end (that stores analysis data), the software can be used to enter, edit, and report information throughout the hazard analysis lifecycle.
