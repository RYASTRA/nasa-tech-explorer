AGATE Simulation Framework

AGATE is a finite-volume scheme framework capable of solving the Hall and CGL (with pressure anisotropy) magnetohydrodynamics equations on multiple distributed processors compatible with both CPUs and Graphics Processing Units (GPUs). The code itself is written in Python and utilizes several open-source libraries to accelerate required calculations. AGATE is being developed as part of the MARBLE project to investigate and model the connection between magnetospheric dynamics and the aurora.
