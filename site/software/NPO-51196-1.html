Python-ION interface (pyion)

Case number
NPO-51196-1
NASA center
JPL
Category
data and image processing
First seen
2026-07-20

pyion provides an Python interface to JPL's Interplantary Overlay Network (ION) via a Proxy class that allows create Endpoints through which you can send or receive bytes, bytearrays and memoryviews. They operate in a manner similar to how TCP/UDP sockets operate.
