Simulation of the Four Bed CO2 Scrubber

The Four Bed CO2 (FBCO2) scrubber simulator is a 1-D mathematical model containing partial differential equations representing mass, heat, and momentum balances within a sorbent bed loaded with zeolite or silica gel sorbent beads. The overall simulation is written in MATLAB, which calls the phenomenological simulation executed in COSMOL for each of the four sorbent beds. Data for each sorbent bed execution is retrieved from COMSOL within Matlab and stored in text files for access by the subsequent COMSOL sorbent bed simulation. The MATLAB code iterates the entire FBCO2 cycle until cyclic steady state conditions are achieved, then outputs plots of the relevent parameters, such as temperatures, partial pressures, and absolute pressures. For validation runs, the plots of simulation data are overlaid with experimental data for similar boundary and initial conditions.
