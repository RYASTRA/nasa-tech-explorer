Hierarchical Data Format Earth Observing System (HDF-EOS) Web Server

Case number
GSC-15011-1
NASA center
GSFC
Category
data servers processing and handling
First seen
2026-07-20

This shell script chains together existing data usability group tools to: extract ODL metadata from an HDF-EOS file; convert the metadata to XML; reformat the XML into HTML; publish the HTML and the original HDF-EOS file to a Web server and an OPeNDAP server; and reformat the XML and submit it to the Earth Observing System Clearing House (ECHO). For more information, please visit: http://opensource.gsfc.nasa.gov/projects/heserve/index.php
