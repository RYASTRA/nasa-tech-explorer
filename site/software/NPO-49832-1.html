Knowledge Acquisition and Synthesis for Information Model Driven Architectures (CornerStone)

Case number
NPO-49832-1
NASA center
JPL
Category
data and image processing
First seen
2026-07-20

This framework uses established requirements and best-practices for long-term digital repositories to create an information model for a set of one or several related domains in a community. The information model is subsequently used to configure the functional components of an information model-driven architecture. The result is an information system what meets the information requirements levied on the system by the community.
