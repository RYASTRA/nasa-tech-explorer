The CHarring Ablator Response (CHAR) Code is a 1D, 2D, and 3D ablation, thermal analysis and porous flow solver primarily used to predict the response of ablative thermal protection systems during atmospheric entry and ground testing. Using the finite-element method, it solves the governing partial differential equations for charring ablators in addition to being a general thermal analysis code. It solves both direct and inverse heat transfer and ablation problems. The software can be executed on serial and parallel computing platforms.
