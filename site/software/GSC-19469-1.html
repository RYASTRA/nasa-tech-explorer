The Astrophysics Cross-Observatory Science Support (ACROSS) system facilitates coordination among multiple astrophysics observatories. The system ingests, normalizes and provides a single point of access to publicly available but siloed NASA mission state and status information (e.g. mission observation plans, orbit parameters, etc.). Using this information, the system provides value-added features, such as multi-mission observing feasibility calculators, decision support functions, target of opportunity (ToO) and generalized mission tasking request forms, enabling users to discover, evaluate and implement coordinated responses to unfolding scientific events. The capabilities of the ACROSS system will be deployed as documented, open-source Application Programming Interface (API) endpoints, and as a software toolkit to facilitate adoption and innovation among NASA missions, general observers, and other science teams. The deployment strategy will allow integrations with third-party APIs (i.e. mission and community users) and facilitate development of custom user-defined applications. The system will also implement a set of user-interface (UI) dashboards hosted on the NASA ACROSS web portal.
