COVER (Calibrate Observed Vegetation Estimates Remotely)

Case number
LAR-19181-1
NASA center
LARC
Category
data and image processing
First seen
2026-07-20

The source code provided develops calibration models using linear regression models with in-situ field data. The calibration models are then used to predict biomass (log), nitrogen percent, and nitrogen content for Landsat images from 2006-2016. Model results and data tables are output as separate files for each field season (i.e. winter and spring seasons).
