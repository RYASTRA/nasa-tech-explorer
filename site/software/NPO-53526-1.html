Europa Clipper Flight System Performance Analysis (FSPA) Downlink Software Development

Case number
NPO-53526-1
NASA center
JPL
Category
aeronautics
First seen
2026-07-20

The software described (both new tools and upgrades to existing tools) facilitates rapid and robust analysis of the spacecraft via downlinked telemetry of various sorts. General capabilities include:- Management and visualization of data used in analysis, including standard channel and EVR data types, as well as other types including planning predicts and derived trends- Provides contextual awareness of data in relation to rule limits- Tracking of onboard FSW parameters and the onboard Ephemeris Table
