Condor is a mathematical modeling framework for engineers with a deadline. Condor provides a modular framework for creating and numerically evaluating engineering models. Model equations are defined using an expressive mathematical domain-specific language implemented using Pythons metaprogramming and an external symbolic backend. When evaluating models, the process of transforming the model equations, creating numerical functions, calling solvers, and extracting the values is fully automated, allowing engineers to focus on engineering rather than coding. Condor includes a variety of built-in model templates, from simple explicit systems and table lookups to more sophisticated algebraic systems, trajectory analyses, and optimization problems. An emphasis was placed on flexibility and modularity, so external tools and solvers can be easily incorporated.
