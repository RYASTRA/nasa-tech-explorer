Building Open-Source Hydrodynamic Models Using ANUGA Software

Case number
NPO-52605-1
NASA center
JPL
Category
data and image processing
First seen
2026-07-20

This software uses an automated approach to build hydrodynamic models of coastal areas. The software is Python based and downloads and incorporates publicly available data to populate model parameters. The software utilizes several common packages as well as an open-source modeling software, ANUGA. ANUGA is available on Github and solves the shallow water equations using a finite volume method over an unstructured mesh. Our software allows the user to build the input files needed for ANUGA, set up and run an ANUGA simulation, and process the output files for visualization and interpretation.
