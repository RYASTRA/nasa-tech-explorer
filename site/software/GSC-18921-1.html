The Limit Checker application is one of the reusable applications that make up the Core Flight System(CFS). Limit Checker monitors telemetry data points in the flight system and compares the values against predefined or computed threshold limits. When a threshold condition is encountered, an event message is issued and a command script may be initiated to respond to the threshold violation.
