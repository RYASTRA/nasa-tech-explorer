NASA World Wind is virtual globe technology provided as an Application Programming Interface-Centeric (API-Centric) Software Development Kit (SDK). It is elemental technology allowing others to see "their" spatial data in the native context of Earth or any other planet for which there is data. Any application, proprietary or other, can use NASA World Wind to see their data in a 3D context.
