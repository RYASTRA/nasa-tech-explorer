NASMAT PrePost: A Visualization and Analysis Tool for the NASA Multiscale Analysis Tool

NASMAT PrePost is an open-source python-based visualization and analysis tool that performs pre-processing of inputs and post-processing of results for the NASA Multiscale Analysis Tool (NASMAT). Users can use the tool to generate or open NASMAT input files, run a NASMAT analysis, and visualize multiscale analysis results through a user interface. Limited support is included for visualizing results obtained via the Abaqus finite element software/NASMAT interface. NASMAT-specific functions are decoupled from the user interface so that a user can generate models independently if desired.NASMAT PrePost is intended to be used alongside with the separately-provided NASMAT software. While most functions of NASMAT PrePost can be utilized without having NASMAT, NASMAT is required if a user wants to generate their own results. NASMAT is also distributed via the NASA Software Catalog.
