This report describes a database routine named DB90 that is intended for use with scientific and engineering computer programs that require relational database functionality. The database software is written in Fortran and conforms to the Fortran 2003 programming language standard.The DB90 functional inter-operability is targeted for scientific and engineering software that also is written in Fortran; however, DB90 is compatible with software written in any language that allows for making calls to executable Fortran subroutines, and should be portable to any computing platform and operating system with a Fortran 2003 compiler.
