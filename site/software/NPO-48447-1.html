pyam is a software development framework with tools for facilitating the rapid development of software in a concurrent software development environment. pyam provides solutions for thorny development challenges associated with software reuse, managing multiple software configurations, developing of software product-lines, and multiple platform development and build management. pyam uses release-early, release-often development cycles to allow developers to incrementally integrate their changes into the system on a continual basis.
