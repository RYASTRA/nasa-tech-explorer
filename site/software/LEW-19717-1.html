Advanced Geared Turbofan 30,000 (AGTF30)

The Advanced Geared Turbofan 30,000 (AGTF30) is a geared turbofan simulation that utilizes the Toolbox for the Modeling and Analysis of Thermodynamic Systems (T-MATS) to create a steady-state and dynamic engine model within MATLAB/Simulink. The engine model is based upon an N+3 geared turbofan concept with a controller that allows dynamic operation throughout the flight envelope.
