Gigatraj is software that implements an atmospheric trajectory model. Gigatraj is intended for use by atmospheric scientists studying the upper troposphere and lower stratosphere.It is written in C++, with a set of classes that allow for users to select data sources, interpolation methods, integration methods,etc. It can be run in serial mode on a single processor, or it can be run in a multiprocessing environment.
