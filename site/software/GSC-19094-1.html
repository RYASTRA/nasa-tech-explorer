ASIST is a real-time command and control system that provides an environment for the development, integration, testing, and on-orbit operation of spacecrafts, their component subsystems and instruments, and external control and support equipment. This NTR is an addendum to the NTR filed for ASIST 22.0.ASIST 23.0 is the first ASIST build on Red Hat Enterprise Linux, version 8 and Ubuntu versions 18 & 20.
