Tool For Interactive Plotting, Sonification, and 3D Orbit Display (TIPSOD)

Case number
GSC-14732-1

GSFC
Category

First seen
2026-07-20

TIPSOD is a software application designed for interactive, animated, 4D (space and time) visualization of satellite orbits. The technology is implemented in Java 3D and is an extension to the existing Satellite Situation Center Web (SSCWeb) 2D static orbit graphics. Please visit the following URL for additional information: http://opensource.gsfc.nasa.gov/projects/tipsod/index.php
