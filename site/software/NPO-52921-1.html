Firstly, planetary magnetic field simulation software is not publicly available for all the planets, and neither are all the various models. This unified software package includes all the giant planets in widely used languages, making it accessible by anyone in the community. To our knowledge, no other publicly available software provides features for inversion of the frequency spectrum of magnetic oscillations experienced along an arbitrary trajectory.
