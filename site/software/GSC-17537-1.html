core Flight Software (cFS) Stored Command Absolute (SCA) Application

Case number
GSC-17537-1
NASA center
GSFC
Category
autonomous systems
First seen
2026-07-20

This software is a core Flight System (cFS) application that allows a system to be autonomously commanded via files that contain sequences of commands. Each command has a time tag associated with it, permitting the command to be released for distribution over the cFS core Flight Executive (cFE) Software Bus at predetermined times. The application allows up to 5 absolute-time sequences to be run simultaneously.
