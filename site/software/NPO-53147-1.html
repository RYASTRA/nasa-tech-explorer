Sorting and PDS4 Labeling of DSN TRK-2-34 Data

Case number
NPO-53147-1
NASA center
JPL
Category

First seen
2026-07-20

All deep space missions performing radio science must archive raw data within the Planetary Data System (PDS). The Tracking and Navigation File (TRK-2-34) is the current standard format for radiometric data acquired at the Deep Space Network (DSN). This software allows users to quickly create the necessary label compliant with the current PDS4 standards for archiving within the PDS.
