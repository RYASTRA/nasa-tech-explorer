DEWPython is a Python implementation of the Deep Earth Water (DEW) model for calculating thermodynamic properties of aqueous species at high temperatures (100-1200 degrees C) and pressures (1-60 kbar). The package streamlines the input process for the DEW model and integrates SUPCRT96/SUPCRTBL functionality, allowing users to compute Gibbs energies of formation, equilibrium constants, and standard volume changes for reactions in extreme environments.
