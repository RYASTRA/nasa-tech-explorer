VERSA: Vortex Event Reactive Sensor Algorithm

Case number
NPO-53599-1
NASA center
JPL
Category
aeronautics
First seen
2026-07-20

This software is designed to develop and test ML algorithm prototypes for detecting events in time series data onboard small planetary spacecraft. Specifically, our package aims to autonomously detect previously recorded Martian dust vortices using time-series data from pressure sensors. This is meant to help research how future spacecraft might be able to use low-power monitoring mode to identify events of interest. Once identified, the spacecraft could then switch to a high-power, high-data-volume observation mode and spend more resources when a scientifically interesting event (e.g., a dust devil) is occuring. This approach aims to enable future technology that can enhance the scientific return from a mission while substantially reducing power consumption and data transmission needs.
