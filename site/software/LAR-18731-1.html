Drought Severity Assessment - Decision Support Tool (DSAT)

Case number

NASA center
LARC
Category
data and image processing
First seen
2026-07-20

The Drought Severity Assessment - Decision Support Tool allows a user to calculate Standardized Precipitation Index (SPI) values for each cell of a set of monthly precipitation rasters that span at least thirty years. Additionally the tool calculates basic SPI statistics within a user specified boundary, and allows for visualization of calculated SPI rasters over time.
