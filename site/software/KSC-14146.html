The NASA Claims Provider is an authentication provider built for Microsoft SharePoint. The NASA Claims provider allows code to be executed once a user authenticates to a SharePoint site. Once a new token is created, events are trigger to claims to the current user's account using claims augmentation. Claims not available in part of the out of the box functionality in SharePoint can be added to a user's token. The solution also allows for customization of people picker actions through the SharePoint infrastructure. A companion application to migrate existing user accounts to a SAML based claims authentication format is provided as part of the NASA Claims provider solution.
