The General Mission Analysis Tool (GMAT) is the world's only enterprise, multi-mission, open source software system for space missiondesign, optimization, and navigation. The system supports missions in flight regimes ranging from low Earth orbit to lunar, libration point, and deep space missions. GMAT can be operated from an interactive Graphical User Interface (GUI), from a custom script language, or from Python or Java programming environments. The GMAT development team consists of NASA, private industry, public, and private contributors and is used for real-world mission support, engineering studies, education applications, and public engagement.
