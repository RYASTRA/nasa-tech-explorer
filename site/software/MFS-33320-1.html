Chemical Equilibrium with Applications in MATLAB (CEAM)

Case number
MFS-33320-1
NASA center
MSFC
Category
propulsion
First seen
2026-07-20

This software is an adaptation of the FORTRAN based Gordon-McBride Chemical Equilibrium with Application (CEAM) computer program developed 60 years ago. This software incorporates modern programming techniques, data management, and adds unique user features. A standalone graphical user interface application has been created: an App for use within MATLAB is provided; a standalone version of the GUI App is also provided. A stand alone executable application is now included. A package for adding CEAM to python has been compiled.
