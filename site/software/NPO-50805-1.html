OpenSCRUB is an orchestration framework that is used to run and aggregate results from multiple static code analysis tools. OpenSCRUB is customizable to the needs of the project, allowing teams to tailor execution to meet their needs. OpenSCRUB also allows users to export static analysis results to Collaborator in order to perform peer reviews.The end goal of OpenSCRUB is to incorporate static analysis results into peer reviews to ensure that all the static analysis findings are addressed and discussed. The internal OpenSCRUB GUI can also be used by developers strictly as a viewer for browsing static analysis results from multiple sources.
