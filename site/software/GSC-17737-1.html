NASA Operational Simulator for Small Satellites (NOS^3)

NOS3 is an open source software only test bed for small satellites. It is a collection of Linux executables and libraries. Current simulations are based on commercial off the shelf (COTS) hardware that is being used on the STF-1 CubeSat. It is intended to easily interface with flight software developed using the NASA Core Flight Software (cFS).
