TrickCFS is a software package that provides the C structs, C++ classes and pertinent code required to synchronize a core Flight System (cFS) mission with the Trick simulation executive. Time is modeled by the simulation and the Trick Memory Manager provides a method to peek, poke, and data record cFS application data without modifying any FSW code.
