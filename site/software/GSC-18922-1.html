The Memory Dwell (MD) application is one of the reusable applications that make up the Core Flight System (cFS). MD provides the capability of dwelling on an address or range of addresses on a flight system which can be very useful tool when trying to diagnose a failure. If the Operating system supports symbolic addressing, Memory Dwell supports specifying the memory address using a symbolic address.
