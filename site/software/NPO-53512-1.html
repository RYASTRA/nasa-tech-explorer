EOS MLS Level 1 Data Processing Software, version 6

Case number
NPO-53512-1
NASA center
JPL
Category
data and image processing
First seen
2026-07-20

This software accepts the EOS MLS Level 0 science/engineering data and the EOS Aura spacecraft ephemeris/attitude data and produces calibrated instrument radiance and associated engineering and diagnostic data. The unique features of this software lie with the instrument and its unique capabilities to measure atmospheric radiance at the earth's limb.
