CLC - a joint extrinsics-only LIDAR-camera calibration tool

Case number
NPO-53445-1
NASA center
JPL
Category
data and image processing
First seen
2026-07-20

CLC establishes a procedure for gathering data to compute the relative transformations between each sensor's coordinates. CLC processes this data to report the sensor geometry and some uncertainty metrics to let the user know how trustworthy the results are, and how this confidence could be improved
