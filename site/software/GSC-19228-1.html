The Front End Data System (FEDS) is a gateway between the user and the spacecraft. It decodes and interprets incoming spacecraft telemetry frames, extracting telemetry packets and delivering them to a number of client machines attached via Transmission Control Protocol/Internet Protocol (TCP/IP). It can also encapsulate and encode your command packets or frames and transmit them to your spacecraft.
