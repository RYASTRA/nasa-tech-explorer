The Fortran Argument Parser is a software package to simplify handling of command line arguments from within Fortran programs. The design is largely based upon a similar package in Python called argParser. These tools allow developers to specify expected command line arguments and associate each argument with a specific behavior. The package is extensible allowing developers to define new behaviors for their options.This package emerged from development of NASA's pFUnit framework for unit testing. The functionality would be valuable within other Fortran projects and therefore it is desired to pull the Fortran Argument Parser out as a software project of its own. The parser can then be used in a variety of projects in a light-weight manner.
