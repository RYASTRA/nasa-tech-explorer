Lunasynth Software

Case number
NPO-53275-1
NASA center
JPL
Category
data and image processing
First seen
2026-07-20

This software creates and renders synthetic terrains of the moon. It enhances a given Digital Elevation Map (DEM) with a statistical distribution model of rocks and craters. It can render the terrain using physics-based lighting.
