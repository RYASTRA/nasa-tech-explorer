The Goddard Mission Services Evolution Center (GMSEC) 5.3 software includes the GMSEC Application Programming Interface (API) along with software applications that enable the automation, situational awareness and verification of ground data systems for space flight missions. The GMSEC software leverages the scalable, open and extensible framework that the GMSEC API enforces through its abstraction of message-oriented middleware and adherence to the Command and Control Message Specification (C2MS) standard managed by the international standards organization Object Management Group (OMG) (https://www.omg.org/spec/C2MS/About-C2MS/). This release of the GMSEC software also includes additional features for the GMSEC API that enable secure communication between services and components operating in a ground system.
