This is a software package for operating the Robotiq Hand-E robotic gripper (https://robotiq.com/products/hand-e-adaptive-robot-gripper), a COTS product, through the Robot Operating System. It is built on top of a BSD licensed open-source driver for other Robotiq grippers from PickNik (https://github.com/PickNikRobotics/ros2_robotiq_gripper).
