Ocean Spatiotemporal Clustering System (OSTCS)

Case number
NPO-53570-1
NASA center
JPL
Category
data and image processing
First seen
2026-07-20

A modular geospatial remote sensing pipeline that integrates NASA satellite data for oceanographic analysis using location blind time series clustering techniques. Features automated data downloading from NASA EarthData and implementation of functional data clustering via hypothesis testing k-means (Zambom et al., 2019) adapted for remote sensing applications. The clustering algorithm uses ANOVA and t-test statistics with conditional logic for time series grouping, processing 10 years of satellite data (May 2015-May 2025) to identify temporal patterns independent of spatial location before mapping results geographically. Built on hybrid Python-R architecture with functional data analysis, multi-resolution data fusion, and visualization capabilities.
