nu-Anomica (Previously Sparse One Class Support Vector Machines (SOC-SVMs))

Case number
ARC-16346-1
NASA center
ARC
Category
data and image processing
First seen
2026-07-20

nu-Anomica is an anomaly detector that can run faster than traditional OC-SVMs. The technology can handle large training sets and works with a well-defined target function. The program utilizes the OSU SVMs code (which is a MATLAB version of Lib-SVMs) as the baseline.
