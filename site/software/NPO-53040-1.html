Priestley-Taylor Jet Propulsion Laboratory Evapotranspiration Model Python Package

This software package is a Python implementation of the Priestley-Taylor Jet Propulsion Laboratory (PT-JPL) model of evapotranspiration. The PT-JPL model was designed for processing remote sensing data. It has the ability to partition latent heat flux into canopy transpiration, interception, and soil evaporation.
