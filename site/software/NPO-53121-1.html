Breathing Earth System Simulator (BESS) Model Python Implementation

This software package is the python implementation of the Breathing Earth System Simulator (BESS) model of gross primary production (GPP) and evapotranspiration (ET). This software makes the BESS model accessible for remote sensing researchers as part of the evapotranspiration modeling capabilities being developed for the ECOSTRESS and SBG missions.
