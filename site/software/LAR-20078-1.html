SUITCASE Structural Analysis and Optimization System

SUITCASE is a Fortran computer code that performs static structural analysis, static aeroelastic loads analysis, aircraft trim, dynamic structural analysis, flutter analysis and structural optimization. For structural analysis, SUITCASE calls the external program EPSM (Equivalent Plate Structural Modeling) and for flutter analysis, SUITCASE calls the external program DLFLUT. All other analysis tools are contained within the SUITCASE system. For optimization, SUITCASE uses the KSOPT optimizer and the LPSOLVE optimizer.
