The airPy package was developed to extract high-resolution satellite data from Google Earth Engine (GEE) and calculate Machine Learning-ready features for air pollution studies. This software prepares raw data from GEE into ML-ready NetCDF files. Extra capabilities such as grid-matching is provided. No other software offers these features, so airPy was developed in-house to enable future researchers easier access to GEE datasets.
