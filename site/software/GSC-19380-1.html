The Evolutionary Mission Trajectory Generator (EMTG) version 9.02 is a scalable-fidelity optimization package for space trajectory design. EMTGv9 is suitable for trade studies, sensitivity analysis, and serves as an initial guess generator for a flight fidelity tool. EMTG can design missions using high-thrust chemical propulsion or low-thrust electric propulsion. Included in this release is the Python EMTG Automated Trade Study Application (PEATSA) for trade analysis.
