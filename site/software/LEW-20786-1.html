The Prologix GPIB-Ethernet Controller Configuration Utility is a piece of software designed to interface with and configure a Prologix GPIB-Ethernet Controller. These controllers are used to communicate between an ethernet and a GPIB interface, and they are often used to connect a data acquisition system to a piece of instrumentation. This software is used to configure the multitude of settings, both network settings and GPIB settings, that exist on these controllers. This tool is a command line interface (CLI) utility that operates as single-use commands (that can be scripted for automation) run from any standard command line to interact with and configure the Prologix hardware on Windows system.
