gpsGUI: EFIS-style GPS and IMU Visualization, Logging and Log-Playback Software

Case number
NPO-52170-1
NASA center
JPL
Category
data and image processing
First seen
2026-07-20

The gpsGUI software provides real-time Electronic Flight Instrument System ("EFIS") style visualization of GPS and IMU data coming over a binary TCP/IP stream. gpsGUI can be used as a stand-alone graphical program, or, various pieces can be incorporated into existing codebases using standard c++ interfaces. The software is written with an emphasis on thread-safe real-time operation, and is designed for critical operations where corrupted or missed data would not be tolerable.
