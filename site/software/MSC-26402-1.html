The Software Toolkit for Engineering and Aeroscience Models (STEAM) enables rapid development and deployment of integrated databases for analysis products. It is written in Python and C++ to provide an API to database and tool developers. The software depends on several open source libraries. It can read and write common file formats used for computational fluid dynamics and aeroscience analysis. No geometry or solution data is included in STEAM. Products created using STEAM can combine tabulated data, surface maps of dependent quantities, and engineering models. STEAM can perform geometric manipulation and spatial interpolation using unstructured meshes and flight space interpolation to relevant design conditions. Data manipulation can be performed during flight space interpolation. STEAM facilities the creation of databases and bespoke tools to interpret them.
