Machine Learning Pressure Probe Calibration Tool

Case number
LEW-20426-1
NASA center
GRC
Category
aeronautics
First seen
2026-07-20

An open source Machine Learning example code designed to educate the public on how to use Machine Learning to retrieve flow direction from a probe. The framework relies on training a machine learning model on normalized calibration data. A test case is presented with examples to show the user how input pressures can be translated to flow angles.
