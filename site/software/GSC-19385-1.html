Ephemeris Library (autoNGC library for managing ephemerides, covariance information, and attitude information)

The ephemeris library provides functionality for storing and querying historical, current, and future ephemeris, covariance, and attitude information. It is implemented as a flight software library capable of running within the core flight system (cFS) as part of the autonomous guidance, navigation and control (autoNGC) suite of applications and libraries.
