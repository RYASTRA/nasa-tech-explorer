Log Truck Weighing System

Case number
HDQS-SO-63
NASA center
HDQS
Category
transportation
First seen
2026-07-20

Eldec Corporation manufactured an instrument in truck cabs that displays electronically computed weight data enabling truck operators to improve earnings by maximizing loads without exceeding legal highway weight limits. Electronic equipment computes tractor weight and this information is presented to driver by an instrument in cab. System costs $2,000 but pays for itself in a single year. Approximately 2,000 logging trucks now use the system.
