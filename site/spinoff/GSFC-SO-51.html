Under Goddard Space Flight Center and Jet Propulsion Laboratory SBIR contracts, AST Engineering Services, Inc. developed a system engineering computer software tool to model how certain application will affect a proposed system's performance. Quantitative System Engineering (QASE) evaluates system timing, capacity and availability. The system is used to predict performance of proposed real-time, aerospace systems, embedded systems, and/or scientific systems, as well as in support of NASA's EDOS initiative.
