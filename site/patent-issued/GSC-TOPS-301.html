Modular Architecture for a Resilient Extensible SmallSat (MARES) Command and Data Handling (C&DH) Hardware

Case number
GSC-TOPS-301
NASA center
GSFC
Category
instrumentation
First seen
2026-07-20

In the quest for faster and cheaper missions, there has been a push to utilize commercial off the shelf components. While these components are designed to operate in a space environment for a year, there is a risk of failure in a single large radiation event. The innovation of a next generation command and data handling (C&DH) board combines off the shelf parts with core radiation tolerant components.
