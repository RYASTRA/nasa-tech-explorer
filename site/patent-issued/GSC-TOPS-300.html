Data obtained from very high resolution (VHR) satellites are difficult to use by scientists for the following reasons. The data is often not in a standard, geographic information system (GIS)-ready format. The data has poor horizontal and vertical co-registration. Once ortho-rectified GeoTiffs are produced, the data can have large file sizes and require High End Computing (HEC) environments to process and analyze many images in an efficient manner. GeoTIFF is a public domain metadata standard which allows georeferencing information to be embedded within a TIFF file. The High-Resolution Commercial Validation Products API assists Earth surface studies with improved data usability.
